Tag, Class and ID Selectors Examples

Class Selectors Example

Different Class (color changed even with h3 tag.)

<h3 class = "red">

1 class can be assigned to different elements.

1 element can have multiple classes.

<h3 class = "red italic">

Class Selectors Example Code

.red {
 color: red;
}
.italic {
 font-style: italic;
}

ID Selectors Example

Class and ID Selectors can overlap. Id tags should be unique. One element cannot have multiple id selectors.

<h3 class="red" id="underline">

Id tag takes priority over class tags.

<h3 class="red" id="brown">

ID Selectors Example Code

#brown {
 color: brown;
}
#underline {
 text-decoration: underline;
}

Id Selectors are generally not recommended.

Pseudoclass Examples

You found me! :hover is an example of pseudoclass. Elements that have pseudoclass represents that they have different "states". (For example, whether the cursor is on the text for :hover.)

Example Code:

.invisible {
 color: #FFF4e4
}
.invisible:hover {
 color: black;
}
