Display Properties

Block: Takes up whole width. Can adjust width.
Inline: Next html element is inline with current html element. Cannot adjust width.
Inline-block: Inline but can adjust width.

Positioning

Static: Follows flow of html elements.
Relative: Positioning based on previous position.
Absolute: Positioning based on parent element. Removes element from html flow.

Centering

  1. text-align: center in parent element. However, it does not work if width is set/changed.
  2. Use margin-left / margin-right: auto or margin: some value 0
